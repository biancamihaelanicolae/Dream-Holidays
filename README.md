# Dream Holidays - Rezerva vacanta de vis

Dream Holidays este o platforma comerciala destinata utilizatorilor care doresc sa isi planifice si sa isi rezerve calatoriile intr-un mod simplu si eficient. Proiectul se concentreaza pe oferirea unei experiente de utilizare intuitive, punand accent pe claritatea informatiilor si pe un design adaptabil.

---

## 1. Impartirea informatiilor pe categorii si subcategorii

Structura continutului este organizata astfel incat sa faciliteze gasirea rapida a serviciilor dorite:

### Destinatii
* **Europa:** Optiuni pentru City break-uri si vacante la plaja.
* **America si Asia:** Propuneri pentru vacante de lunga durata (minim 2 saptamani).
* **Locatii exotice:** Pachete speciale pentru Insule si Safari.
* **Munte:** Oferte pentru drumetii si activitati de schi.

### Servicii
* **Cazare:** Selectie de hoteluri, apartamente in regim hotelier si resorturi.
* **Pachete de activitati:** Excursii cu ghid si bilete pentru atractii turistice.

### Suport clienti
* Gestionarea rezervarilor.
* Sectiune de intrebari frecvente (FAQ).
* Formular de contact.

---

## 2. Identificarea paginilor si a legaturilor dintre ele

Arhitectura site-ului cuprinde urmatoarele pagini fundamentale:

1. **Pagina Home (Acasa):** Prezentare generala, bara de cautare, sectiuni pentru "Oferte Top" si "Destinatii Populare".
2. **Pagina Destinatii (Listing):** Vizualizarea locatiilor sub forma de carduri, cu filtre pentru buget si servicii.
3. **Pagina Detalii Vacanta:** Informatii complete despre pachete, galerie foto si sistem de selectie a perioadei.
4. **Pagina Pachete Speciale (Experiente):** Dedicata ofertelor care includ activitati si bilete la obiective turistice.
5. **Pagina Contact:** Contine formularul oficial de comunicare.

**Navigarea:** Interconectarea paginilor se realizeaza prin meniul principal si butoane de actiune de tip "Vezi Detalii".

---

## 3. Cuvinte si sintagme cheie (SEO)

Proiectul utilizeaza termeni relevanti pentru optimizarea cautarilor:
* **General:** vacanta, calatorie, rezervari online, pachete turistice, oferte hoteluri.
* **Pagina Home:** destinatii 2026, oferte last minute, cauta vacante.
* **Pagina Detalii:** facilitati hotel, recenzii, rezervari imediate.

---

## 4. Analiza site-urilor similare

Am studiat competitia pentru a identifica avantajele si dezavantajele tehnice si de design:

| Site | Pro | Contra |
| :--- | :--- | :--- |
| **Booking.com** | Filtrare avansata, recenzii verificate | Interfata aglomerata, mesaje de presiune |
| **Tripadvisor.com** | Integrare cu restaurante, poze reale | Navigare greoaie, multe reclame |
| **Expedia.com** | Pachete Zbor + Hotel, design curat | Check-out lung, incarcare lenta pe mobil |
| **DirectBooking.ro** | Oferte specifice pietei locale | Design vizual invechit, filtre limitate |

## Tehnologii Utilizate
* **HTML5** - Structura semantica a paginilor.
* **CSS3** - Layout bazat pe Grid si Flexbox pentru design responsive.
* **Font Awesome** - Integrarea elementelor grafice de tip iconita.
* **Google Fonts** - Utilizarea fonturilor Montserrat si Playfair Display.
